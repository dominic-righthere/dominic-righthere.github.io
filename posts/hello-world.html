Welcome to my blog! This is my first post written in MDX, which allows me to combine Markdown with React components.

Why MDX?

MDX gives you the best of both worlds:

  1. Write content in Markdown (easy to read and write)
  2. Embed React components for interactivity
  3. Use custom styling for any element
