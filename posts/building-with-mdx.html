How It Works

  1. Create your React component in the /components directory
  2. Import it at the top of your MDX file
  3. Use it like any other JSX element
import { TemperatureSlider } from "@/components/interactive/temperature-slider";

// Then use it in your content:
<TemperatureSlider />

This makes your blog posts much more engaging and informative!
